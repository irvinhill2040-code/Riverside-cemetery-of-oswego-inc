One-time cloud setup — takes about 5 minutes

Connect to Supabase (free)

This gives every device — phones in the field, office computers, member devices — access to the same live records and headstone photos.

1
Go to supabase.com and create a free account
2
Click New Project, name it riverside-cemetery, choose a region closest to New York
3
Once created, go to Project Settings → API and copy your Project URL and anon/public key
4
Go to SQL Editor in Supabase and run the setup SQL (tap "Copy SQL" below, paste it there, click Run)
5
Paste your URL and key below and tap Connect
